# PYTHON-CODEZ

Collection of Python programs and examples.

This repository contains small Python scripts, practice problems, and mini projects used to learn and practice Python programming.

## Contents
- Basics: loops, conditionals, input/output
- Functions and modules
- Data structures and algorithms
- Mini projects and practice problems

## How to run
1. Clone the repo:
   git clone https://github.com/shubham-nov06/PYTHON-CODEZ.git
2. Open the folder in your IDE.
3. Run a file:
   python filename.py

## Requirements
- Python 3.x
